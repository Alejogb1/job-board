Here's a code snippet  that shows you how to use GPT-3  to generate some creative content

```python
import openai

openai.api_key = "YOUR_API_KEY"

response = openai.Completion.create(
  engine="text-davinci-003",
  prompt="Write a poem about a robot falling in love with a human",
  max_tokens=1024,
  temperature=0.5
)

print(response.choices[0].text)
```
