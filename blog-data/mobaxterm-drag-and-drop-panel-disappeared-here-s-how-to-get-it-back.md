---

```
# In MobaXterm, the drag and drop panel is not displayed. How to fix it?
# First, go to the "Settings" menu.
# Then, select "Preferences".
# Under "General" > "Session Management", check the "Enable Drag and Drop" checkbox.
# Click "OK" to save your settings.
# You should now see the drag and drop panel.
```
